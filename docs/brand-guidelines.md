## 1. 品牌定位

### 1.1 品牌故事

广澜印刷包装有限公司深耕印刷包装行业，现开放闲置厂房空间，为同行企业提供专业、可靠的生产场地租赁服务。品牌传达的核心信息是：**懂印刷行业的厂房租赁**——不只是出租空间，更是为印刷包装同行提供真正适用、配套齐全的生产车间。

### 1.2 品牌人格

| 特质 | 描述 |
|------|------|
| **专业** | 深耕印刷行业，了解厂房需求（层高、承重、供电等） |
| **可靠** | 真实厂房、真实参数、清晰定价 |
| **务实** | 不花哨，信息透明，流程简单 |

## 2. 色彩系统

### 2.1 设计理念

灵感来源：**印刷 CMYK + 工业厂房**

- **Teal（深青）** = 印刷油墨 Cyan 的演变，代表印刷行业基因
- **工业灰蓝** = 钢结构、混凝土的现代工业质感
- **暖琥珀** = 厂房灯光/木托盘的暖色，作为唯一暖色点缀

## 3. 字体系统

### 3.1 字体堆栈

```css
--font-heading: 'Noto Sans SC', 'Inter', system-ui, -apple-system, sans-serif;
--font-body: 'Noto Sans SC', 'Inter', system-ui, -apple-system, sans-serif;
--font-mono: 'JetBrains Mono', 'Fira Code', monospace;
```

## 4. Logo 概念

### 4.1 设计方案

**主 Logo 概念：** "广澜"二字 + 厂房轮廓几何图形

| 元素 | 设计 |
|------|------|
| 图形标记 | 厂房抽象几何图形 — 用简洁线条勾勒厂房轮廓（坡屋顶 + 立柱），暗合印刷 CMYK 四色方块概念 |
| 中文标准字 | "广澜租赁" — Noto Sans SC Bold，深青 #0F766E |
| 英文副标 | "GUANGLAN RENTAL" — Inter Medium，工业灰蓝 #334155，字母间距 0.15em |
| 组合方式 | 左图右字（横式）或上图下字（竖式） |

### 4.2 Logo 变体

| 变体 | 用途 |
|------|------|
| 全横式（图标 + 中文 + 英文） | 网站导航栏、文档抬头 |
| 竖式（图标 + 中文） | 页脚、移动端 |
| 图标独立 | Favicon、社交媒体头像 |
| 单色版（工业灰蓝） | 水印、合同文件 |

### 4.3 安全空间

Logo 周围最小留白 = 图标高度的 50%，确保在任何背景下呼吸感充足。

## 5. 品牌语音

### 5.1 语音特征

| 维度 | 我们是 | 我们不是 |
|------|--------|----------|
| 专业 | 懂印刷厂房的行家 | 只会收租的房东 |
| 清晰 | 参数真实、价格透明 | 含糊其辞、全是面议 |
| 务实 | 解决问题、快速响应 | 画大饼、过度承诺 |
| 稳重 | 值得信赖的合作伙伴 | 高高在上、冷冰冰 |

### 5.2 场景语调

| 场景 | 语调 | 示例 |
|------|------|------|
| 首页 Hero | 大气、专业 | "印刷人的厂房，就该懂你。" |
| 厂房详情 | 精确、全面 | "层高 6.5m，承重 3t/㎡，供电 250kVA" |
| 下单引导 | 清晰、友好 | "选择租赁周期，30 秒提交意向" |
| 后台界面 | 高效、直接 | "今日新增 3 条待审核订单" |
| 邮件通知 | 正式、完整 | "您的租赁订单 GW-20260707-001 已通过审核" |

### 5.3 禁用语

| 避免 | 原因 |
|------|------|
| 顶级/一流/领先 | 空洞自夸 |
| 白菜价/跳楼价 | 与 B2B 专业调性不符 |
| 100%/绝对/保证 | 过度承诺 |

## 7. 图片风格

### 7.1 厂房摄影

- **光线**：自然光为主，展示厂房真实采光
- **角度**：广角展示空间全貌 + 局部展示地面/屋顶/配套设施细节
- **色调**：保持真实，可微调增加对比度，不添加滤镜
- **构图**：横平竖直，突出空间感和结构

### 7.2 图标

- 风格：Outlined 线框风格（Lucide Icons / Heroicons）
- 线宽：1.5px
- 圆角：2px
- 尺寸：24px 基准
