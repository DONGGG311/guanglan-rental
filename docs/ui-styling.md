## 页面索引

| 页面 | 路由 | 关键特征 |
|------|------|----------|
| 首页 | `/` | Hero 搜索 + 分类入口 + 推荐厂房 + CTA |
| 厂房列表 | `/spaces` | 筛选侧边栏 + 卡片网格 + 分页 |
| 厂房详情 | `/spaces/[id]` | 图片轮播 + 参数网格 + 下单入口 |
| 用户中心 | `/user/*` | 侧边导航 + 订单/收藏/通知 |
| 后台管理 | `/admin/*` | 深色侧边栏 + 数据表格 + 表单 |

### Hero 区域

| 属性 | 值 |
|------|------|
| 背景 | 渐变色：`bg-gradient-to-br from-teal-700 to-teal-900` |
| 高度 | min-h-[480px]（移动端 min-h-[360px]） |
| 标题 | text-5xl（桌） / text-3xl（移）、font-bold、text-white |
| 副标题 | text-lg、text-teal-100、mt-4、max-w-2xl、mx-auto |

```

- 右侧搜索按钮 `bg-accent`（琥珀色）、rounded-lg、font-medium

```

### 筛选侧边栏（桌面端 240px，移动端 Drawer）

| 区域 | 样式 |
|------|------|
| 容器 | `w-60`、`bg-white`、`border-r border-slate-200`、`p-5`、`sticky top-16` |
| 分组标题 | text-sm font-semibold text-slate-800 mb-3 |
| 复选框项 | flex items-center gap-2 text-sm text-slate-600 py-1 |
| 单选 | accent-teal-700（checkbox color） |
| 重置按钮 | text-sm text-slate-400 hover:text-slate-600 |

### 筛选区 — 移动端

```
┌──────────────────────────────┐
│ 筛选  ▼  │ 共 12 套厂房      │
└──────────────────────────────┘
```
- 点击"筛选"弹出 Drawer（从左侧滑入）
- Drawer 内布局同桌面侧边栏

### 卡片网格

| 属性 | 值 |
|------|------|
| 容器 | `grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-6` |
| 卡片 | SpaceCard 组件（见设计系统） |
| 过渡 | hover:translateY(-4px) transition duration-200 |

### 排序栏

```
┌──────────────────────────────────────┐
│ 共 12 套厂房  │ 默认排序 ▼          │
└──────────────────────────────────────┘
```

### 空状态

当筛选无结果时：
```
         🏭
   未找到匹配的厂房
  尝试调整筛选条件
     [清除筛选]
```

| 属性 | 值 |

## 全局样式规范

### 页面容器

```css
.container { max-width: 1280px; margin: 0 auto; padding: 0 24px; }
/* 移动端: padding: 0 16px; */
```

### 链接

- 默认：`text-teal-700 hover:text-teal-800`
- Navbar：`text-slate-600 hover:text-teal-700`

### 分割线

- `border-t border-slate-100`（浅）
- `border-t border-slate-200`（标准）

### Loading 状态

- 骨架屏：`bg-slate-200 animate-pulse rounded`（Card/Table/Text）

### 过渡统一

```css
transition-all duration-200
```

### 滚动条（可选）

```css
::-webkit-scrollbar-thumb { background: var(--color-slate-300); border-radius: 4px; }
```
