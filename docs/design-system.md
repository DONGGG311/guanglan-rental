## 1. SpaceCard（厂房卡片）

列表页和首页的核心展示单元。

### 布局

```
┌─────────────────────────────┐
│                             │
│      [厂房图片/占位图]        │  ← 16:9，object-cover
│                             │
├─────────────────────────────┤
│ 小型厂房          🟢 可租    │  ← AreaBadge + AvailBadge
│                             │
│ 厂房名称                     │  ← text-lg, font-semibold
│ 📍 地址摘要                  │  ← text-sm, text-secondary
│                             │
│ ├ 150㎡  ├ 月租 ¥8,000      │  ← 关键参数行
│                             │
│          查看详情 →          │  ← link/hover 上移阴影
└─────────────────────────────┘
```

### 规格

| 属性 | 值 |
|------|-----|
| 宽度 | 可变（Grid: 1fr, minmax 320px） |
| 圆角 | `var(--card-radius)` 12px |
| 默认阴影 | `var(--shadow-sm)` |
| 悬停阴影 | `var(--shadow-md)` |
| 内边距 | `var(--card-padding)` 24px |
| 图片区 | 无 padding，延伸到顶边 |
| 过渡 | `var(--duration-normal)` 200ms |

### 状态

| 状态 | 表现 |
|------|------|
| 默认 | 白底，浅阴影 |
| 悬停 | 上浮 4px（translateY(-4px)），阴影加深 |
| 已租 | 图片区叠加 "已出租" 半透明遮罩 |

## 4. 按钮系统（Buttons）

| 类型 | 背景 | 文字色 | 边框 | 用途 |
|------|------|--------|------|------|
| Primary | `teal-700` | white | 无 | 主要操作（提交、保存） |
| Secondary | transparent | `teal-700` | `teal-700` | 返回、取消 |
| CTA/Accent | `amber-600` | white | 无 | 立即租赁、立即咨询 |
| Ghost | transparent | `slate-500` | 无 | 取消、低调操作 |
| Destructive | `red-600` | white | 无 | 删除、拒绝 |

### 尺寸

| 尺寸 | 高度 | 内边距 X | 字号 |
|------|------|----------|------|
| sm | 32px | 12px | 14px |
| default | 40px | 16px | 14px |
| lg | 48px | 24px | 16px |

### 状态

| 状态 | 表现 |
|------|------|
| Default | 正常色 |
| Hover | 加深一档（teal-700 → teal-800） |
| Focus | 2px 外环 `ring-teal-400/50` |
| Disabled | 透明度 50%，cursor: not-allowed |
| Loading | 左侧 spinner + 文字变淡 |
